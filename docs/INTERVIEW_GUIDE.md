# Developer interview guide (Phase 0)

Target: 10 developers who use Claude Code or Cursor on real projects weekly.

## Questions

1. What frustrates you most about using AI coding tools on multi-day projects?
2. Have you lost track of what you were building between sessions? How do you recover context?
3. Do you know how much you spend on API calls per month? Would real-time budget warnings help?
4. How often do you catch hallucinated imports, wrong APIs, or invented functions?
5. Would you pay ~$29/month for persistent project memory + progress tracking + import validation?
6. What is the **one** feature that would make you install a tool on day one?
