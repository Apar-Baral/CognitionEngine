# Cognition Engine

**AI Development Orchestrator**

*Early development — APIs and commands may change.*

## Vision

AI coding tools forget everything between sessions, hallucinate imports and APIs, and offer little cost control. Cognition Engine fixes this with **persistent memory**, **phased planning**, **validation**, and **token budgeting**.

## Quick start

```bash
pip install -e ".[dev]"
cc --help
cc init
cc start
cc end
cc status
```

## Tech stack

Typer, Rich, httpx, ChromaDB, tiktoken, NetworkX, SQLite metrics, YAML config, jsonschema validation.
